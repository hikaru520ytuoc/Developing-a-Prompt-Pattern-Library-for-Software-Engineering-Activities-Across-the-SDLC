# Prompt Output Template

Dùng file này để kiểm tra kết quả đầu ra của AI sau khi chạy prompt.

```text
Tên giai đoạn:
Mã prompt:
Ngày tạo:
Phiên bản:

1. Tổng quan
2. Input đã sử dụng
3. Giả định
4. Kết quả chính
5. Bảng/tài liệu được tạo
6. Checklist đánh giá
7. Điểm còn thiếu/mơ hồ
8. Việc cần làm tiếp theo
```
