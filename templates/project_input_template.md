# Project Input Template

Dùng file này để chuẩn bị thông tin đầu vào trước khi chạy từng prompt.

```text
1. Tên dự án:
[Điền tên dự án]

2. Mô tả ngắn gọn:
[Mô tả hệ thống cần xây dựng]

3. Bối cảnh/vấn đề:
[Mô tả vấn đề hiện tại]

4. Đối tượng sử dụng/actor:
[Liệt kê actor]

5. Mục tiêu:
[Mô tả mục tiêu nghiệp vụ, người dùng, kỹ thuật]

6. Chức năng dự kiến:
[Liệt kê chức năng]

7. Yêu cầu phi chức năng:
[Hiệu năng, bảo mật, usability, maintainability...]

8. Business rules:
[Quy tắc nghiệp vụ nếu có]

9. Công nghệ dự kiến:
[Frontend, backend, database, deployment...]

10. Ràng buộc:
[Thời gian, nhóm, môi trường, yêu cầu báo cáo...]

11. Mức độ chi tiết mong muốn:
[Cơ bản / trung bình / chi tiết cho báo cáo]
```
