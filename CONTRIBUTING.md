# Contributing

Cảm ơn bạn quan tâm đến SDLC Prompt Pattern Library.

## Cách đóng góp

Bạn có thể đóng góp bằng cách:

- Sửa lỗi diễn đạt.
- Thêm ví dụ cho từng giai đoạn.
- Bổ sung checklist.
- Bổ sung prompt ngắn gọn hoặc prompt chuyên sâu.
- Thêm phiên bản tiếng Anh.
- Thêm ví dụ áp dụng cho dự án khác.

## Quy tắc đóng góp

Khi thêm prompt mới, hãy giữ cấu trúc:

```text
Tên prompt
Mã prompt
Mục tiêu
Input cần cung cấp
Prompt đầy đủ
Output mong muốn
Checklist đánh giá
Lỗi cần tránh
```

## Quy ước đặt tên file

```text
NN_short_name.md
```

Ví dụ:

```text
10_ai_code_review_prompt.md
```
